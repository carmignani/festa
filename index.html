Welcome to FESTA webpage!

FESTA is an electron spin tracking code based on the matlab Accelerator Toolbox.

You can download the code from the link on top of the page.

Read this nice paper about the simulations done with FESTA on the ESRF storage ring and on the Australian Synchrotron storage ring: https://jacowfs.jlab.org/conf/y15/ipac15/prepress/MOPWA013.PDF
